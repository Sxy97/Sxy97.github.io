搭博客心路历程

自从开始学代码，就想着建个个人博客玩，从后台JavaSE、JavaEE，到前台html、css、js。一路学过来，过程虽然漫长但收获很多。在本地用Java开发个博客，开发个简单的商城都是可以办到的了，但已经没有布到服务器上的念头了。自己总结有两点原因：一是linux不熟，每次操作linux头疼。二是服务器有点贵啊一年好几百，头疼。结果在现在的实习公司学node.js又让我燃起了搭博客的念头，我感觉这跟hello word一样，是学每个新东西必做的事。接下来就遇到了hexo，没有linux,还不用花钱买服务器，界面还这么好看，一定是特别的缘分。花了半天时间搭起来，又花了二十块钱买了个域名，hexo就是这么简单，半天网站上线了。虽然自己没有码代码，hexo写博客的方式有点费劲，但这总归实现了当初的愿望，有了一个个人博客。

初学node.js感受

当初是拒绝从Java转node.js的，但后来在北京漂了半个月，面了几次试，才知道学Java的人是真多，我这渣渣学校还没毕业想找个实习有点难啊。so就到这儿公司转node了。node会js有后端思维是很好上手的。学习一周了说一下自我感受：
1.node建站很快，Java的时候SSM框架都得配会儿。
2.因为es6语法还没有灵活掌握，所以回调函数用起来很痛苦。
3.要用异步的思维去写代码，node的精髓事件驱动。
4.改变了对js的认知

对未来的一些想法

Java可以node，前端也能转node，如果只以上手开发算转node成本很低。所以对未来的方向：
1.深入了解js,掌握es6
2.掌握node框架，express,koa,egg
3.熟练 node mongoose 建站到 打包上线，整个流程
4.vue react 接触前端框架，打通前后端

全栈是一种信仰，跨前后端，跨网页，android，ios。
